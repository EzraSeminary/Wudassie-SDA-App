# Wudassie App

A comprehensive worship app featuring complete collections of Seventh Day Adventist Amharic Hymnals and Hagerigna songs.

## Features

- 📖 **Complete Hymnal Collection** - Full SDA Amharic Hymnal with lyrics
- 🎵 **Hagerigna Songs** - Dedicated collection of Hagerigna hymns
- ❤️ **Favorites** - Save and quickly access your favorite songs
- 🎼 **Sheet Music** - View sheet music for supported hymns
- 🎧 **Audio & Music** - Access audio recordings and YouTube links
- 🎨 **Customizable** - Adjustable font sizes and dark mode
- 🔍 **Smart Search** - Quick number-based search with modern interface

## Technology Stack

- React Native 0.78.0
- TypeScript
- Redux Toolkit
- React Navigation
- NativeWind (Tailwind CSS)
- Axios

## Installation

```bash
# Install dependencies
npm install --legacy-peer-deps

# iOS
cd ios && pod install && cd ..

# Start Metro bundler
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Requirements

- Node.js 18.x or 20.x (LTS)
- React Native 0.78.0
- iOS 13+ / Android API 21+
